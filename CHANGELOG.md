# Changelog

Все заметные изменения BOOK WUNDERWAFFE Studio фиксируются в этом файле.

## [1.1.0] — 2026-07-22

### Added

- нативная PySide6/Qt WebEngine оболочка для Vue-редактора;
- живой FFmpeg waveform в итоговом MP4;
- движущаяся полоса общего прогресса и таймкод;
- QR-код Telegram в предпросмотре и рендере;
- соседние главы с opacity 50%;
- автоматический подбор размера длинных заголовков;
- мягкий blur и glitch-обработка фона;
- чтение геометрии и visibility слоёв из `editor-project.json`;
- CPU-fallback при отказе аппаратного кодировщика;
- 60-секундный лимит тестового экспорта без потери полного контекста глав.

### Changed

- продукт переименован в **BOOK WUNDERWAFFE Studio**;
- версии API, desktop shell и Vue-пакета синхронизированы на `1.1.0`;
- waveform и progress больше не запекаются в неподвижный PNG;
- заголовок экспорта приведён к размеру Vue-сцены;
- для NVENC и AMF заданы целевой и максимальный видеобитрейт;
- мастер-аудио теперь кодируется один раз после video-only склейки;
- число параллельных тяжёлых workers ограничено двумя;
- прогресс render job обновляется по этапам подготовки и числу завершённых сегментов.

### Fixed

- конфликт `-vf` и `-filter_complex`, ломавший live-waveform export;
- немой предпросмотр аудиокниги;
- Windows launcher и пути с кириллицей;
- потеря соседних глав в тестовом рендере;
- статичная полоса прогресса, зависевшая от номера главы;
- чрезмерный битрейт AMF/NVENC для многочасовых видеокниг;
- возможные AAC-зазоры на границах глав;
- запуск интерфейса во внешнем браузере вместо GUI.
